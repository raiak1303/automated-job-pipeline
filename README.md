
# 🎯 Automated Job Search Pipeline

A Python pipeline that automatically fetches, filters and emails 
job opportunities daily from 18+ sources worldwide.

## Features
- 18+ job sources (LinkedIn, Bayt, Reed, Adzuna, GulfTalent, and more)
- 30+ countries including Gulf, Europe, Singapore, Australia, Canada
- Daily automated email with direct apply links
- Excel report with all opportunities
- Recruiter name and contact info where available

## Tech Stack
Python • BeautifulSoup • SQLite • openpyxl • smtplib • APScheduler

## Setup
1. Clone the repo
2. pip install -r requirements.txt
3. Add your API keys in config.py
4. Run: python pipeline.py
